### This project demonstrates how to use the modern ShareUI dialog, a built-in Windows feature that allows you to send selected files or folders directly to other people, nearby devices, or installed apps (like Mail, Skype, or messaging platforms).

It triggers the native Windows sharing panel, offering the following capabilities:

- Nearby Sharing: Sends files wirelessly to nearby PCs or mobile devices using Bluetooth or Wi-Fi.
- App Sharing: Instantly attaches the selected file to a new email in your default client or sends it through other compatible apps.
- Cloud Sync: Depending on your setup, it often integrates with OneDrive, allowing you to generate shareable links or manage cloud permissions directly.In modern Windows versions, you may see this as an icon at the top of the right-click menu or as a standard text option.
